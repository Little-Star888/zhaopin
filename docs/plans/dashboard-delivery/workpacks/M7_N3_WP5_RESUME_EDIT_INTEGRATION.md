> 预估改动量：修改 ~10 行（修复联调问题）

## 1. 前置条件

- M7-N3-WP4 通过（前端双模式编辑就绪）

## 2. 联调检查项

- [ ] 上传简历 → 后端解析 → 前端 GET 获取 content_md → 显示结构化 HTML

### 2.2 边界情况

- [ ] content_md 为空时的前端降级显示

| 场景 | 后端行为 | 前端期望 |
|------|---------|---------|
| GET /api/resume，content_md 为 NULL | 返回 content_md: null | 降级显示"暂无简历内容" |
