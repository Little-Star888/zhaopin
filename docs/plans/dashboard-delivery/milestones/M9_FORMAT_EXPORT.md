## 宏观交付边界

- **Markdown 为唯一事实源（SSOT）**：编辑态只认 `content_md`，HTML/DOCX/PDF 都是派生输出
- PDF 是输出终点，不参与回环编辑链
- DOCX 承担"可编辑高保真"角色，PDF 承担"高保真最终投递"角色
- 完美转换定义：语义完整 + 可读版式稳定 + 输出体积可控（不追求像素级一致）
- 文件大小上限：5MB
